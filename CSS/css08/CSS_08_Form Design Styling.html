CSS 폼 디자인 & 스타일링(Form Design & Styling)

사용자의 입력을 받아 서버에 전송하는데 사용하는 폼 디자인을 스타일링에 대해 알아봅니다.
폼 디자인 시, 사용자가 입력된 내용의 상태를 이해하고 대처할 수 있도록 고려해 설계해야 합니다.

문제는 폼 컨트롤 디자인은 매우 까다롭다는 점이며, 기술적인 이유로 CSS로 폼 컨트롤을 완벽하게 스타일링할 수 없습니다.

폼 스타일링 요소

CSS를 사용해 완벽하게 스타일링 할 수 있는 요소들

  • <form>
  • <fieldset>
  • <label>
  • <output>

CSS를 사용해 완벽하게 스타일링 할 수 없는 요소들

  • <legend>
  • placeholder 속성

CSS를 스타일링이 전혀 적용되지 않는 요소들

  • <select>
  • <optoion>
  • <optgroup>
  • <datalist>
  • <progress>
  • <meter>
