window 객체

window 객체 메서드 종류

alert()

alert() 메서드는 경고 창을 띄울 때 사용합니다.
window.alert() 메서드는 window 객체를 따로 작성하지 않아도 사용할 수 있습니다.

기본형 : alert("경고 메서지")

  • 경고 창은 화면에 확인 버튼을 클릭해야 alert() 메서드의 다음에 나오는 코드를 실행합니다.

prompt()

질의응답 창을 뛰울 때 사용합니다.
window.prompet() 메서드 앞에 window 객체를 따로 작성하지 않아도 사용할 수 있습니다.

기본형 : prompt("질의 내용","기본 답변");

  • 질문 창을 띄워 유도하는 창이 나타납니다.
  • 확인 버튼을 클릭하면 입력한 답변으로 반환합니다.

confirm()

확인/취소 창을 띄울 때 사용합니다.
window.confirm() 메서드는 window 객체를 따로 작성하지 않아도 사용할 수 있습니다.

기본형 : confirm("질의 내용")

  • 질문 창을 띄워 확인/취소를 누르도록 유도합니다.
  • 이때 확인 버튼 클릭하면 true를 반환하고, 취소 버튼을 클릭하면 false를 반환합니다.

moveTo()

브라우저 창의 위치를 이동시킬 때 사용합니다.(단, 크롬, 오페라 브라우저에서는 정상적으로 작동하지 않습니다.
moveTo() 메서드는 스크린(모니터) 왼쪽을 기준으로 x의 위치값과 위쪽을 기준으로 y의 위치값을 입력합니다.

기본형 : moveTo(x의 위치값, y의 위치값);

resizeTo()

브라우저 창의 너비와 높이를 바꿀 때 사용합니다.(단, 크롬, 오페라 브라우저에서는 정상적으로 작동하지 않습니다.

기본형 : resizeTo(너비값, 높이값);

setInterval()

setInterval() 메서드는 일정한 시간 간격으로 스크립트 실행문을 반복해 실행 시킬 때 사용합니다.
일정한 간격에 자동으로 사진이 바뀌는 사진첩을 만든다거나 자동으로 흐르는 슬라이드 배너를 만들 때 사용합니다.

기본형 : var 참조 변수=setInterval("스크립트 실행문",시간 간격);

clearInterval()

clearInterval() 메서드는 setInterval() 메서드를 사용해 일정한 시간을 간격으로 계속 반복해 실행되는 것을 취소하고자 할 때 사용합니다.

기본형 : clearInterval(참조 변수);

  • 질문 창을 띄워 확인/취소를 누르도록 유도합니다.
  • 이때 확인 버튼 클릭하면 true를 반환하고, 취소 버튼을 클릭하면 false를 반환합니다.

setTimeout()

setTimeout() 메서드는 일정한 간격으로 스크립트 실행문을 한 번만 실행 시킵니다.
setTimeout() 메서드도 재귀 함수 호출을 이용하면 setInterval() 메서드처럼 사용할 수 있습니다.

기본형 : var 참조 변수=setTimeout("스크립트 실행문",시간 간격);

clearTimeout()

clearTimeout() 메서드는 setTimeout() 메서드를 사용해 일정한 시간을 간격으로 한 번 실행할 예정인 실행문을 취소하고자 할 때 사용합니다.

기본형 : clearTimeout("스크립트 실행문",시간 간격);

  • 질문 창을 띄워 확인/취소를 누르도록 유도합니다.
  • 이때 확인 버튼 클릭하면 true를 반환하고, 취소 버튼을 클릭하면 false를 반환합니다.
