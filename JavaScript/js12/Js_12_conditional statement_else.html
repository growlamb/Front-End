조건문(conditional statement) else문

else문

else문은 조건식을 true할 경우와 false 경우에 따라 실행되는 실행문이 달라집니다.
조건식 만족 여부에 따라 수행하는 실행문이 달라지는 else의 기본형입니다.

else문 기본형1

if(조건식) {
실행문1;
} else{
실행문2;
}

else문 예시1

var num=prompt("당신이 좋아하는 숫자는?", "0");
if(nem%2==0){
document.write("당신이 좋아하는 숫자는 짝수입니다.");
}else {
document.write("당신이 좋아하는 숫자는 홀수입니다.");
}

  • 방문자에게 좋아하는 숫자를 입력받습니다. 기본값은 0입니다.
  • 짝수일 경우에 실행
  • 홀수일 경우에 실행
  • 변수 num을 2로 나눴을 때 나머지 값이 0인 경우는 짝수이므로 "당신이 좋아하는 숫자는 짝수입니다."가 출력됩니다.
    0이 아닌 경우는 조건식이 false가 되므로 else 중괄호에 실행문인 "당신이 좋아하는 숫자는 홀수입니다."가 출력됩니다.
