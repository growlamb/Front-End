캔버스(canvas) 요소

원 그리기 메서드

arc(x, y, radius, starAngle, endAngle, antiClockwise) -(x, y)를 중심으로 하여 반지름이 radius인 원을 그립니다.
star는 시작 각도, end 종료 각도입니다.

  • arc() 메서드에서 각도는 상수 PI를 이용합니다.
  • 원은 starAngle에서 endAngle까지만 그려집니다.
  • antiClockwise는 반시계 방향인지 아니면 시계방향인지를 true, false를 이용해 나타냅니다.
  • 원호를 그린 후에 시작점과 종료점을 연결해 그리고 싶으면 closePath()를 호출합니다.
    closePath()는 경로를 닫게 됩니다.

예시

var canvas=document.getElementById("myCanvas");
var context=canvas.getContext("2d");
context.beginPath();
context.arc(100, 100, 80, 0, 2.0*Math.PI, false);
context.strokeStyle= "red";
context.stroke();
context.beginPath();
context.arc(100, 100, 60, 0, 1.5*Math.PI, false);
context.closePath();
context.strokeStyle= "blue";
context.stroke();
context.beginPath();
context.arc(100, 100, 40, 0, 1.5*Math.PI, false);
context.strokeStyle= "green";
context.stroke();
