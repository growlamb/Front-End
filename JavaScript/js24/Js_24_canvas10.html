캔버스(canvas) 요소

이미지로 그리기

캔버스에 이미지를 그리고 위해 메서드를 사용합니다.

  • drawImage(image, x, y) : 화면 (x, y) 위치에 이미지를 그립니다.

이미지를 캔버스에 그리기위해선 drawImage() 메서드를 사용합니다.
drawImage()는 image객체와 좌표를 필요로 합니다.
좌표는 이미지의 좌측 상단이 위치하는 곳입니다.

drawImage() 메서드가 객체를 요구하기 때문에 먼저 image 객체를 생성한 후에 drawImage()를 호출하기 전에 이미지가 로드되기를 기다립니다.
이것은 image 객체의 onload 속성을 이용하면 됩니다.

이미지로 그리기 예시

var canvas=document.getElementById("myCanvas");
var context=canvas.getContext("2d");
var image=new Image();
image.src="logo.png"; image.onload=function() {
context.drawImage(image, 0, 0);};
