캔버스(canvas) 요소

HTML5에서는 이차원 변환으로 평행이동, 신축, 회전이 있습니다.
도형 변환은 일반적으로 행렬을 이용해 자신이 원하는 변환을 생성해 적용할 수 있습니다.

평행 이동(Translation)

평행 이동은 캔버스를 수평이나 수직으로 이동할 수 있습니다. Canvas 클래스 translation() 메서드는 캔버스 좌표 공간의 원점을 평행을 이동합니다.
통상적으로 그래픽스에서 평행 이동과 같은 모든 변환은 내부적으로 행렬을 사용합니다.
(x, y) 좌표에 1을 추가한 동차 좌표를 만든 후에 행렬식을 이용해 변환된 좌표를 계산합니다.

예시

var canvas=document.getElementById("myCanvas");
var context=canvas.getContext("2d");
context.fillStyle="blue";
context.fillRect(0, 0, 100, 100);
context.translate(50, 50);
context.fillStyle="red";
context.fillRect(0, 0, 100, 100);

회전(Rotation)

좌표 공간을 회전시키려면 rotate()를 사용합니다. rotate(float degree)는 (0,0)을 기준으로 좌표 공감을 회전시킵니다.
만약 다른 위치를 기준으로 회전시키려면 rotate(float degree, float px, float py)를 사용합니다. 그러므로 (px, py가 회전의 중심점이 됩니다.

context.translate(50, 50);
context.rotate(Math.PI/4);

신축(Scaling)

캔버스의 신축 비율을 설정하려면 scale() 메서드를 사용합니다.

context.translate(50, 50);
context.scale(0.5, 0.5);

일반적인 변환

일반적인 변환은 직접 변환 행령의 값을 지정하면 됩니다. transform() 메서드를 사용하고 매개 변수 a에서 f까지를 직접 설정하면 됩니다.
context.transform(1, 0, 0, 1, 50, 50);
