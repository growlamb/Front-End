캔버스(canvas) 요소

캔버스는 canvas 요소를 생성합니다.
캔버스는 HTML 페이지상에서 사각 형태 영역을 말합니다.
canvas 요소는 디폴트 값으로 경계를 가지지 않습니다.

canvas id="myCanvas" width="300" height="300"

  • 항상 id속성을 지정합니다.
    id속성이 있어야 자바스크립트에서 참조를 합니다.
  • 가로, 세로 크기 설정합니다.
    경계를 추가할 땐 style 속성을 사용합니다.
    •

    예시

    < canvas id="myCanvas" width="300" height="300" style="border: 1px dotted blue"></canvas>
    var canvas=document.getElementById("myCanvas");
    var context=canvas.getContext("2d");
    context.fillStyle="#0000FF";
    context.fillRect(0, 0, 100, 50);

    • 캔버스 요소를 정의한다.
    • 먼저 캔버스 요소를 찾는다.
    • 캔버스에서 컨텍스트를 얻는다.
    • 캔버스 위에 사각형을 그린다.

    var canvas = document.getElementById("myCanvas");

    HTML 문서 중에서 canvas 요소를 찾는다.

    var context = canvas.getContext("2d")

    캔버스의 getContext() 메서드는 호출해서 컨텍스트 객체를 얻는다. 이때 인수로 "2d"를 전달합니다.
    2d는 2차원 그래픽입니다.

    context.fillStyle = "#0000FF";

    fillStyle속성은 채우기 속성을 정의합니다. 채우기 속성은 색사이나 그라디언트 또는 패턴이 될 수 있습니다.

    context.fillRect(0, 0, 100, 50);

    fillRect(x, y, width, height) 메서드는 현재의 채우기 속성을 가지고 사각형을 화면에 그립니다.
    fillRect() 메서드는 (0, 0, 100, 50) 을 인수로 가지는데 이것의 의미는 좌측 상단에서 시작해 100X50 픽셀의 사각형을 그리는 의미입니다.
