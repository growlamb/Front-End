캔버스(canvas) 요소

텍스트 그리기

캔버스에 텍스트를 그리기 위해서는 다음과 같은 속성과 메서드를 사용합니다.

  • font : 텍스트를 그리는데 사용되는 폰트 속성을 정의합니다.
  • fillText(texxt,x,y) : 캔버스에서 채워진 텍스트를 그립니다.
  • strokeText(text,x,y) : 캔버스에 텍스트의 외곽선만을 그립니다.

예시

이번 예시는 30 pixel 높이의 Arial 폰트로 태워진 텍스트를 그리는 코드입니다.

var canvas=document.getElementById("myCanvas");
var context=canvas.getContext("2d");
context.beginPath();
context.font='italic 38pt Arial';
context.fillText('Hi Hello', 20, 100);
