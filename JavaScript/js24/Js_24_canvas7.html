캔버스(canvas) 요소

선 속성 그리기

선을 그릴 때도 여러가지 속성을 지정할 수 있습니다.

  • context.lineWidth : 선의 두께를 지정합니다.
  • context.strokesStyle : 선의 색상을 지정합니다.
  • context.lineCap : 선의 양쪽 끝점의 모양을 지정합니다.
    butt, round, square 속성을 가지고 있습니다.

예시

이번 예시는 30 pixel 높이의 Arial 폰트로 태워진 텍스트를 그리는 코드입니다.

var canvas=document.getElementById("myCanvas");
var context=canvas.getContext("2d");
context.beginPath();
context.lineWidth=20;
context.lineCap="round";
context.moveTo(20, 20);
context.lineTo(200, 20);
context.stroke();
