캔버스(canvas) 요소

도형 단일 색상으로 채우기

도형의 단일 색상으로 채우려면 다음과 같이 컨텍스트의 fillStyle을 설정합니다.

context.fillStyle='red';

도형 그라디언트로 채우기

그라디언트도 사각형이나, 원, 텍스트를 채우는 데 사용합니다.

  • createLinearGradient(x, y, x1, y1) : 선형 그라디언트를 생성합니다.
  • creatRadialGradient(x, y, r, x1, y1, r1) :원형 그라디언트를 생성합니다.

그라디언트 객체가 생성되면 2개 이상의 종료 색상을 반드시 추가해야합니다.
addColorStop() 메서드는 종료 색상과 위치를 지정합니다.
그라디언트 위치는 0과 1 사이의 실수로 지정됩니다.

선형 그라디언트 예시

var canvas=document.getElementById("myCanvas");
var context=canvas.getContext("2d");
var gradient=context.creatLinearGradient(0, 0, 200, 0);
gradient.addColorStop(0, "white");
gradient.addColorStop(1, "red");
context.fillStyle=gradient;
context.fillRect(10, 10, 180, 90);

도형 그라디언트 예시

var canvas=document.getElementById("myCanvas");
var context=canvas.getContext("2d");
var gradient=context.createRadialGradient(70, 50, 10, 80, 60, 120);
gradient.addColotStop(0, "white");
gradient.addColotStop(1, "red");
context.fillStyle=gradient;
context.fillRect(10, 10, 180, 90);
