캔버스(canvas) 요소

직선 그리기

캔버스에는 직선, 곡선, 사각형, 원, 이미지, 텍스트를 그릴 수 있습니다.

직선을 그리기 위해 메서드
  • moveTo(x, y)는 직선의 시작점을 정의합니다.
  • lineTo(x, y)는 직선의 끝점을 정의합니다.

예시

< canvas id="myCanvas" width="300" height="300"></canvas>
var canvas=document.getElementById("myCanvas");
var context=canvas.getContext("2d");
context.beginPath();
context.moveTo(0, 0);
context.lineTo(100, 100);
context.lineTo(150, 50);
context.lineTo(200, 100);
context.stroke();

  • 경로를 형성해서 그릴 때는 마지막에 항상 stroke()을 호출해야 그림이 실제 그려집니다.
