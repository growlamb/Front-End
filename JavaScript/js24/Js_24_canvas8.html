캔버스(canvas) 요소

패턴으로 채우기

패턴으로 캔버스를 채우기 위해서 createPattern()을 이용해 패턴 객체를 생성합니다.
다음으로 컨텍스트의 fillStyle속성을 패턴 객체로 설정합니다.
최종적으로 fill() 메서드를 호출해서 도형의 내부를 패턴으로 채우면 됩니다.

createPattern() 메서드는 이미지 객체와 repeat 옵션을 요구합니다.
옵션은 repeat(), repeat-x, repeat-y, no-repeat 입니다.
지정되지 않으면 repeat 옵션은 디폴트로 repeat로 설정됩니다.
x, y방향으로 모두 반복하라는 의미가 됩니다.

패턴 패우기 예시

var canvas=document.getElementById("myCanvas");
var context=canvas.getContext("2d");
var image=new Image();
image.src="patter.png";
image.onload=function(){
var pattern=context.createPattern(image, "repeat");
context.rect(0, 0, canvas.width, canvas.height);
context.fillStyle=pattern;
context.fill();};
