자바스크립트(JavaScript)

JavaScript 는 동적으로 업데이트되는 컨텐트를 만들고, 멀티미디어를 제어하고, 이미지를 애니메이션화하는 등 스크립팅 언어입니다.

JavaScript 일반적인 프로그래밍 기능 구성

  • 유용한 값을 변수 안에 저장합니다. 예를 들어 우리는 새로운 이름을 입력 한 다음 그 이름을 호출 된 변수에 저장하도록 요청을 name으로 합니다.
  • Operations on pieces of text(프로그랭미에서 "문자열"이라고 합니다.)에 대해 정의합니다. 예를 들어 문자열 "Player 1 :"을 가져 와서 name변수에 결합하여 완전한 text label을 만듭니다.
  • 웹 페이지에서 발생하는 특정 이벤트에 대한 응답을 이벤트를 사용해 버튼을 클릭 한 시간을 감지합니다. 다음 text label을 업데이트하는 코드를 실행합니다.
