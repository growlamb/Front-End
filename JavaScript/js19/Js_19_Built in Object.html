내장 객체(Built in Object)

내장 객체란?

자바스크립트 엔진에 내장되어 있어 필요한 경우 생성해서 사용할 수 있습니다.
내장 객체로는 문자, 날짜, 배열, 수학, 정규표현 객체 등이 있습니다.

내장 객체 생성하기 기본형

참조 변수(인스턴스 네임) = new 생성함수()

  • 객체 생성할 때에는 new라는 키워드와 생성함수를 사용합니다.

내장 객체 예시

var com=new Object();
com.width="40cm";
com.height="35cm";
com.weight="25kg";
com.color="white";
com.off=function(){
document.write("전원 off","<br />");
}
document.write(com.width,"<br />");
document.write(com.height,"<br />");
document.write(com.weight,"<br />");
document.write(com.color,"<br />");
com.off();

  • 생선된 객체를 변수 com가 참조합니다.
  • com 객체의 속성을 생성합니다.
  • com 객체의 off 기능을 생성합니다.
  • com의 너비 속성을 출력합니다.
  • com의 높이 속성을 출력합니다.
  • com의 무게 속성을 출력합니다.
  • com의 컬러 속성을 출력합니다.
  • com에 off 기능을 실행합니다.
