내장 객체(Built in Object)-날짜 객체(Date Object)

날짜 객체란?

날짜나 시간 관련 정보를 제공받고 싶을 때는 날짜 객체를 생성합니다.
날짜 정보 객체는 날짜와 관련된 작업을 할 때 유용한 객체입니다.
날짜 정보 객체를 이용하면 달력과 특정 기념일 D-day 계산기도 만들 수 있습니다.

Date 객체 기본형1

참조 변수= new Date();

  • 현재 날짜 정보를 제공하는 Date 객체입니다.

Date 객체 기본형2

참조 변수=new Date("연/월/일");
참조 변수=new Date(연, 월-1, 일);

  • 현재 날짜가 아니라 특정 날짜의 정보를 제공받아야하므로 특정 날짜 정보 객체를 만들어 사용합니다.

날짜 객체 메서드

날짜 정보 객체에는 날짜의 정보를 가져오는 메서드와 날짜를 수정하는 메서드로 나누어집니다.

날짜 정보를 가져올 때(GET) 날짜 정보를 수정할 때(SET)
getFullYear() 연도 정보를 가져옴 setFullYear() 연도 정보만 수정
getDate() 일 정보를 가져옴 setDate() 일 정보만 수정
getDay() 요일 정보를 가져옴(일:0 ~ 토:6) 요일은 날짜를 바꾸면 자동으로 바뀌므로 setDay는 없음
getHours() 시 정보를 가져옴 setHours() 시 정보만 수정
getMinutes() 분 정보를 가져옴 setMinutes() 분 정보만 수정
getSeconds() 초 정보를 가져옴 setSeconds 초 정보만 수정
getMilliseconds() 밀리초 정보를 가져옴(1/1000초 단위) setMilliseconds() 밀리초 정보만 수정
getTime() 1970년 1월 1일부터 경과된 시간을 밀리초로 표기함 setTime 1970년 1월 1일부터 경과된 시간을 밀리초로 수정함
toGMTString GMT 표준 표기 방식으로 문자형 데이터로 반환함 toLocaleString 운영 시스템 표기 방식으로 문자형 데이터로 반환함

내장 객체 예시

var a=new Date();
var nowMonth=a.getMonth();
var nowDate=a.getDate();
var nowDay=a.getDay();
document.write("현재 월:"+nowMonth,"<br />");
document.write("현재 일:"+nowDate,"<br />");
document.write("현재 요일:"+nowDay,"<br />");
var i=new Date(2018,1,9);
var theMonth=i.getMonth();
var theDate=i.getDate();
var theDay=i.getDay();
document.write("2018평창올림픽 몇 월:"+nowMonth,"<br />");

  • 현재의 월의 정보를 반환하며, 현재 월보다 한달 적게 나옵니다.
  • 현재 일자의 정보를 반환합니다.
  • 현재 요일의 정보를 반환합니다.

  • 날자 객체에서 요일 정보만 가져옵니다.
