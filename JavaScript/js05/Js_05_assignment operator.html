대입 연산자(Assignment Operator)

대입 연산자란 무엇일까?

대입 연산자는 연산된 데이터를 최종적으로 변수에 저장할 때 사용됩니다. 복합 대입 연산자(+=, -=, *=, /=, %=)란 산술 연산자와 대입 연산자가 복합적으로 적용된 것을 말합니다.

대입 연산자의 종류

종류 풀이
A=B A=B
A+=B A=A+B
A*=B A=A*B
A/=B A=A/B
A%=B A=A%B

대입 연산자 예시

var num1=10;
var num2=3;
num1 += num2;
document.write(num1, "
");
num1 -= num2;
document.write(num1, "
");
num1 *= num2;
document.write(num1, "
");
num1 %= num2;
document.write(num1, "
");

  • 변수 num1에 10을 저장합니다.
  • 변수 num2에 3을 저장합니다.
  • num1 = num1+num2; num1에 저장된 13;
  • num1에 저장된 값을 출력합니다.
  • num1 = num1-num2; num1에 저장된 값 10;
  • num1 = num1*num2; num1에 저장된 값 30;
  • num1 = num1%num2; num1에 저장된 값 0;
