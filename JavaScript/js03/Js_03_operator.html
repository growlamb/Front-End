연산자(Operator)

연산자란 무엇일까?

연산자는 자바스크립트 프로그래밍에서 사용하는 연산자에는 산술, 문자, 문자 결합, 대입(복합 대입), 증감, 비교, 논리, 삼항 조건 연산자가 있습니다.

산술 연산자의 종류와 기본형

종류 기본형 설명
+ A+B 더하기
- A-B 빼기
* A*B 곱하기
/ A/B 나누기
% A%B 나머지

산술연산자

var num1=15;
var num2=2;
var result;
result=num1+num2; //더하기
document.write(result, "
");
result=num1-num2; //빼기
document.write(result, "
");
result=num1*num2; //곱하기
document.write(result, "
");
result=num1/num2; //나누기
document.write(result, "
");
result=num1%num2; //나머지
document.write(result, "
");
