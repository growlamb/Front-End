증감 연산자

증감 연산자란 무엇일까?

증감 연산자는 숫자형 데이터를 1씩 증가시키는 증가 연산자(++)가 있으며, 반대로 1씩 감소시키는 감소연산자(--)가 있습니다. 증감 연산자는 앞에서 배운 연산자와 달리, 피연산자가 한 개만 필요한 단항 연산자입니다.
증감 연산자는 변수의 어는 위치에 오는가에 따라 결과값이 달라집니다.

증감 연산자의 기본형

  • 변수--; 또는 --변수;
    1 감소시킵니다.
  • 변수++; 또는 ++변수;
    1 증가시킵니다.
  • var A=++B;
    변수 B의 데이터를 1 증가시킨후 변수 A에 저장합니다.
  • var A=B++;
    변수 A에 변수 B의 데이터를 저장한 후 변수 B의 데이터를 1 증가시킵니다.

증감 연산자 예시

var num1=10;
var num2=20;
num1 += num2;
var result; num1 --;
document.write(num1, "
");
num1++;
document.write(num1, "
");
result=num2++; document.write(result, "
");
result=++num2;
document.write(num1, "
");

  • 변수 num1에 10을 저장합니다.
  • 변수 num2에 20을 저장합니다.
  • 변수 result를 선언하였습니다.
  • 변수 num1에 데이터가 1감소됩니다.
    변수 num1에 변환된 데이터를 출력합니다.
  • 변수 num1에 데이터가 1증가됩니다.
    변수 num1에 변화되 데이터를 출력합니다.
  • 대입 연산자가 먼저 실행되고, 증가 연산자가 실행됩니다.
    변수 result에 저장된 값 20이 출력됩니다.
  • 증가 연산자가 먼저 발생되고, 대입 연산자가 실행됩니다.
    변수 result에 저장된 값 22가 출력됩니다.
