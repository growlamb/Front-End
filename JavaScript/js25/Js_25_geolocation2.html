위치 정보(Geolocation)

getCurrentPosition() 메서드

위치 정보를 가져올 때 사용하는 메서드입니다. 위치 저오를 처리하는 함수를 인수로 전달합니다. function getLocation(){ var geolocation=navigator.geolocation; geolocation.getCurrentPosition(showLocation, erroHandler); } * showLocation()은 위치 정보를 처리하는 콜백 메서드입니다. * erroHandler()는 오류를 처리하는 콜백 메서드입니다. 오류를 처리하는 메서드는 옵션이므로 없어도 사용이 가능합니다.

showPosition()

function showLocation(position){ var latitude=position.coords.latitude; var longitude=position.coords.longitude; } * position 인수가 위치 정보를 가지고 있는 객체입니다. postion인수는 Position객체의 변수입니다. Position객체는 많은 속성을 가지고 있습니다.

Position 속성

속성 타입 설명
coords objects 장치의 위치 정보를 가지고 있는 객체이다.
coords.latitude Number 위도 정보, 단위는 도(decimal degree)이다. 범위는 [-90.00, +90.00]
coords.longitude Number 경도 정보, 단위는 도(decimal degree)이다. 범위는 [-180.00, +180.00]
coords.altitude Number 고도 정보, 단위는 미터이다.
coords.heading Number 북쪽으로부터 시계방향으로 방위각이다.
coords.speed Number 장치의 현재 속도이다.
timestamp date 위치 정보가 반환된 시각이다.
