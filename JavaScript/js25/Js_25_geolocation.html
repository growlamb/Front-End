위치 정보(Geolocation)

위치 정보는 자신의 위치를 웹 사이트와 공유하는 것입니다. 우리가 위치를 공개하면 지도에서 자신의 위치를 볼 수 있으며 현재 지역의 날씨, 유명한 맛집 등의 정보를 제공받을 수 있습니다. 위치 정보 API는 사용자의 지리적 위치를 가져오는 데 사용됩니다. 위치 정보는 사용자의 개인 정보이기 때문에 사용자가 승인하지 않으면 자바스크립트에서 가져갈 수 없습니다. 위치 정보는 navigator객체가 가지고 있는 geolocation 객체를 통해 얻을 수 있습니다. 위치 정보를 가지고 있는 변수 var geolocation = navigator.geolocation;

메서드 설명
getCurrentPosition() 사용자의 현재 위치 정보를 반환한다.
watchPosition() 장치의 현재 위치에 대한 정보를 주기적으로 반환한다.
cleatWatch() 현재 진행 중인 watchPosition() 실행을 중지한다.
