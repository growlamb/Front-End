반응형 웹 페이지 디자인(Responsive Web Design)

PC용 웹 사이트는 스마트폰이 등장하면서 웹 디자인 영역에도 큰 영양을 주었습니다. 스마트폰 화면에 맞춘 사이트를 따로 만들어애 했습니다.
반응형 웹이 등장하면서 PC, TV, 내비게이션, 스마트 기기 등 다양한 기기에 반응하는 웹사이트입니다.
사용자가 접속하는 기기 또는 환경에 맞춰 크기가 자동으로 조덜되어 최적화된 화면 레이아웃을 보여줍니다.

반응형 웹 페이지 디자인 장점

  1. 유지관리가 편합니다.
  2. 최신 웹표준으 따릅니다.
  3. 기기의 화면 크기나 해상도에 구애받지 않습니다.
  4. 가로, 세로 화면의 변화에 즉각 반응합니다.
  5. 어느 기기에서든 사이트 주소가 같습니다.

단점으로는 예전 브라우저 버전과 호환성에 문제가 있을 수 있으며 사이트 디자인이 단순해집니다.
