SVG(Scalable Vector Graphics)

HTML는 인라인 svg를 지원합니다. SVG는 웹에서 벡터기반의 그래픽을 정의하는데 사용되고 상호작용과 애니메이션도 지원합니다.
장점으로는 벡터 그래픽은 직선, 곡선, 원, 사각형과 같은 도형을 이용해 그래픽을 정의하는 방법입니다.
또한, 확대, 크기 변경되어도 품질이 손상되지 않으며 svg 이미지는 검색이 가능하고 인덱싱 스크립트화 압축이 가능합니다.

SVG로 도형그리기

  1. 원 그리기
    circle cx="100" cy="100" r="50" stroke="black" stroke-width="3" fill="red"
  2. 사각형 그리기
    rect width="300" height="100" style="fill:rgb(255,0,0); stroke-width: 3; stroke: rgb(128,128,128)"
  3. 타원 그리기
    ellipse cx="100" cy="50" rx="100" ry="50" style="fill:red; stroke:black; stroke-width:2"
  4. 직선 그리기
    line x1="0" y1="0" x2="200" y2="100" style="fill:none; stroke:red; stroke-width:6"
  5. 폴리라인 그리기
    polyline points="10,10 150,20 180,70 230,80" style="stroke:red; stroke-width:3"
  6. 다각형 그리기
    polygon points="100,20 250,160 60,210" style="fill:red; stroke:black; stroke-width:2"
  7. 텍스트 그리기
    text x="20" y="30" font-family="Arial" font-size="25" fill="red"
