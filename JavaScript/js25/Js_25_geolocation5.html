위치 정보(Geolocation)

이동 중 위치 정보를 얻는 방법

사용자가 이동하면서 주기적으로 위치 정보를 얻으려면 다른 방법을 사용합니다. geolocation 객체의 watchPosition()을 호출해 콜백 메서드를 등록합니다. 콜백 메서드가 주기적으로 호출되면서 업데이트된 위치 정보를 얻을 수 있습니다. clearWatch()를 호출해야만 주기적인 호출을 중지합니다.

  • watchPosition() - 사용자의 현재 위치를 연속해 출력합니다. 자동차처럼 사용자가 이동하고 있으면 계속 업데이트된 위치를 반환합니다.
  • clearWatch() - watchPosition() 메서드를 중지 시킵니다.

예시

var id; var myDiv=document.getElementById("target"); function starGeolocation() { if(navigator.geolocation) { id=navigator.geolocation.watchPosition(showGeolocation); } } function stopGeolocatio() { if(navigator.geolocation) { navigator.geolocation.clearWatch(id); } }
