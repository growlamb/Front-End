조건문(conditional statement) if문

조건문이란 무엇일까?

조건문은 조건식의 값이 true인지 아니면 false인지에 따라 실행문의 제어가 결정됩니다.

if문

if문 조건식을 만족(true)할 경우에 실행문을 실행합니다.

if문 기본형

if(조건식) {
실행문;
}

if문 예시

var num=10;
if(num<500){
document.write("hello");
}

  • 조건을 만족하여 true를 반환합니다.
  • num<500의 비교결과는 10<500이므로 true를 반환합니다.
    조건문 내의 실행문을 실행하면 "hello"가 출력됩니다.
