지역 변수와 전역변수

전역변수란?

함수 정의문 내에서 var를 붙이지 않는 변수이며, 이 변수에 저장된 데이터는 자바스크립트가 선언된 곳이라면 사용할 수 있습니다.

지역변수란?

함수 정의문 내에서 var를 붙인 변수로, 이 변수에 오는 데이터는 함수 정의문 내에서만 사용할 수 있습니다.


전역변수 기본형

var 변수;
function 함수명(){
변수=값;
}

지역변수 기본형

function 함수명(){
var 변수=값;
}

  • 전역과 지역 변수를 나누는 이유는 변수의 중복 사용을 피하기 위함
  • 함수 정의문 내에서 var를 붙여 지역변수를 이용하게 되며 여러 함수에 같은 변수가 있어도 중복될 일이 없습니다.
  • 하지만 한 개의 변수를 여러 함수에 공용해 사용하고 싶으면 전역 변수를 선언해야 합니다.

예시

var num=100;
function myFnc(){
var num=400;
}
myFnc();
document.write(num);

  • 가장 상단에 있는 num은 전역변수이며, 함수 정의문 내에 변수는 var가 있으므로 지역 변수입니다.
  • 함수에 실행문으로 var를 붙여 작성한 변수는 지역 변수이므로 함수의 밖에서 불러와 사용할 수 없습니다.
  • 함수 밖에 선언된 num의 변수의 값인 100이 출력됩니다.
