매개 변수가 있는 함수 정의문

기본 함수 정의문이란?

함수 내에 있는 실행문을 실행시킬 때 단순히 함수명으로 호출해 실행문을 실행합니다.
함수를 호출할 때 값을 전달 할 수 없으며 매개변수가 있는 함수정의문은 함수를 호출할 때 전달 값을 입력해 호출 할 수 있습니다.

함수를 호출할 때 값을 전달하는 기본형

function 함수명{(매개 변수 1, 매개 변수 2,...매개 변수 n) 스크립트 실행문; } 함수명(데이터 1, 데이터 2,...데이터 n);

  • 함수 호출 시 데이터 1은 매개 변수 1에 저장됩니다.
  • 데이터2는 매개 변수2에 저장됩니다.
  • 그러므로 차례대로 데이터는 매개 변수에 저장됩니다.

예시

function myFnc(name, area({
document.write("반갑습니다."+name+"입니다.","<br />");
document.write("사는곳은"+area+"입니다.","<br /><br />");
}
myFnc("김철수","서울");
myFnc("박영희","부산");

  • 매개 변수가 있는 함수
  • 김철수는 name에 서울은 area에 전달합니다.
  • 박영희는 name에 부산은 area에 전달합니다.

내장 함수

내장함수란 자바스크립트 엔진에 내장된 함수 정의문을 말합니다.
내장함수는 함수 정의문의 선언 없이 단지 함수 호출만으로 자바스크립트에 이미 내장된 함수들을 사용 할 수 있습니다.

내장 함수의 종류

종류 설명 예시
parselnt() 문자형 데이터를 정수형 데이터로 바꿉니다. parselnt("5.12")->5
parseFloat() 문자형 데이터를 실수형 데이터로 바꿉니다. parseFloat("5.12")->5.12
String() 문자형 데이터로 바꿉니다. String(5)->"5"
Number() 숫자형 데이터로 바꿉니다. Number("5")->5
Boolean() 논리형 데이터로 바꿉니다. Boolean(5)->true
isNaN() 데이터에 숫자가 아닌 문자를 포함하면 true를 반환합니다. isNaN("5-3")->true
isNaN("53")->false
eval() 문자형 데이터를 큰따음표가 없는 스크립트 코드로 처리합니다. eval("15+5")->20
