return문

return문 이란?

함수에서 결과값을 되돌려 줄 때 사용됩니다.
함수 정의문에서 실행문으로 return문이 사용되었으면 함수를 호출했을 때 결과값을 반환합니다.

데이터를 반환하는 return 기본형

function 함수명(){
실행문;
return 데이터(값);
}
var 변수=함수명();

예시

function myFnc(a, b){
var num=a*b;
return num;
}
var result=myFnc(10,3);
document.write(result);

  • 함수 호출문에 30을 반환합니다.
  • 함수 호출 시 데이터가 매개 변수에 전달되어 함수 내에 있는 실행문에 사용됩니다.
  • 계산 결과를 return문으로 함수 호출문에 되돌려 주어 변수 result에 저장해 문서에 출력하였습니다.

강제 종료 역할을 하는 return문

함수가 호출되면 함수 정의문 내에 실행문이 실행됩니다.
이때 return문이 실행되면 return문 다음에 오는 실행문은 실행되지 않고 강제로 종료됩니다.

function 함수명(){
실행문
return;
실행문;
}
함수명();

예시

function myFnc(){
document.write("hello");
return;
document.write("javascript")
}
myFnc();

  • return문을 사용해 함수에 실행문을 강제로 종료시켜 "javascript"문구를 출력되지 않습니다.

재귀 함수 호출

최초 함수를 호출하면 함수 정의문 내에서 실행문으로 함수를 다시 호출하는 것을 재귀 함수 호출입니다.
함수를 반복문 처럼 여러 번 호출하기 위해 사용합니다.

재귀 함수 기본형

function myFnc(){
실행문;
myFnc();
}
myFnc();

예시

var num=0;
function testFnc(){
num++;
document.write(num,"<br />");
if(num==10) return;
testFnc();
}
testFnc();

  • num의 데이터가 1씩 증가됩니다.
  • 재귀함수 호출 testFnc함수를 호출합니다.
  • testFnc함수를 호출합니다.
  • testFnc에 실행문으로 재귀 함수 호출문은 삽입해 반복적으로 testFnc가 호출됩니다.
  • 조건문을 사용해 재귀 함수 호출을 종료시켰습니다.
