숫자나 문자 입력 검증

사용자가 신용카드 번호, 전화번호, 우편번호와 같은 정보를 입력하면 웹 페이지는 이드리 모두 숫자로 되어 있다는 것을 검증합니다.
숫자로만 되어 있는 데이터를 검증하는 가장 좋은 방법은 정규 표현식을 사용하는 것입니다.
/^[0-9]+$/은 문자열이 모두 숫자로만 되어 있어야 일치합니다.

script

function checkNumeric(elem, msg) {
var exp=/^[0-9]+$/; if (elem.value.match(exp)) {
return true;
} else {
alert(msg);
elem.focus();
return false;
}}

form

전화번호(-없이 입력): <input type='text' id='num' />
<input type='button' onclick="checkNumeric(document.getElementById('num'), '숫자만 입력하세요')" value='확인' />
