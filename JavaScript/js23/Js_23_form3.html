정규식

정규식이란 특정한 규칙을 가지고 있는 문자열을 표현하는 수식입니다.
정규식은 많은 텍스트 에디터와 프로그래밍 언어에서 문자열의 검색과 치환을 위해 사용됩니다.

정규식 메타문자

식 기능 설명
^ 시작 무자열의 시작을 표시
$ 끝 문자열의 끝을 표시
. 문자 한 개의 문자와 일치
/d 숫자 한 개의 숫자와 일치
/w 문자와 숫자 한 개의 문자나 숫자와 일치
/s 공백 문자 공백, 탭, 줄 바꿈, 캐리지 리턴 문자와 일치
[] 문자 종류, 문자 범위 [abc]는 a 또는 b 또는 c를 나타낸다.

정규식 수량 한정자

수량 한정자는 문자가 몇 번 반복하는지 나타냅니다.

수량 한정자 기능 설명
* 0회 이상 반복 a*는 ", a, aa, aaa를 나타냅니다.
+ 1회 이상 a+는 a, aa, aaa를 나타냅니다.
? 0 또는 1회 a?는 ", a를 나타냅니다.
{m} m회 a{3}는 aaa만 나타냅니다.
{m, n} m회 이상 n회 이하 a{1, 3}는 a, aa, aaa를 나타냅니다.
(ab) 그룹화 (ab)*은 "", ab, abab 등을 나타냅니다.
