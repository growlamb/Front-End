선택 검증

HTMl select 필드에서 사용자가 선택을 했는지 검증하려면 살짝의 트릭이 필요합니다.
첫 번째 옵션을 도움말로 두는 것입니다.
ex) "선택하세요"라는 문구를 제시하고 만약 사용자가 선택을 하지 않았다면 이 문구가 반환 될 것입니다.

script

function checkSelection(elem, msg) {
if (elem.value == 0) {
alert(msg);
elem.focus();
return false;
} else {
return true;}}

form

과일 선택 <select id="fruits" class="required">
<option value="0">선택하세요.</option>
<option value="1">딸기</option>
<option value="2">귤</option>
<option value="3">오렌지</option>
<input type='button' onclick="checkSelection(document.getElementById('fruits'), '하나를 선택해주세요')" value='확인' />
