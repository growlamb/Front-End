중첩 if문

중첩 if문은 조건문 안에 조건문이 오는 것을 말합니다.

중첩 if문 기본형

if(조건식1) {
if(조건식2){
실행문;
}}

  • 중첩 if문은 일반적으로 바깥쪽에 있는 조건문인 조건식 1을 만족해야만 안쪽에 있는 조건문인 조건식2를 확인합니다.
  • 만일 안쪽 if문의 조건식2를 만족하지 않은다면 바깥쪽 조건문인 조건식 1의 중괄호 내에 있는 실행문만 실행하고 종료됩니다.

중첩 if문 예시

이번 예시는 중첩 if문을 사용해 아이디와 비밀번호가 일치했을 때 환영 문구가 출력됩니다.
비밀번호가 일치하지 않을 경우네는 브라우저가 새로고침이 실행됩니다.

var id="admin";
var pw="1111";
var user_id=prompt("아이디는?", "");
var user_pw=prompt("비밀번호는?", "");
if(id==user_id){
if(pw==user_pw){
document.write(user_id+"님 반갑습니다.");
}else{
alert("비밀번호가 일치하지 않습니다.");
location.reload();
}else{
alert("아이디가 일치하지 않습니다.");
location.reload();
}

  • 방문자에게 아이디를 입력받습니다.
  • 방문자에게 비밀번호를 입력받습니다.
  • 아이디가 일치했을 경우
  • 비밀번호가 일치하기 않을 경우
  • 브라우저 새로고침이 실행됩니다.
  • 안쪽 if문의 else문 종료
  • 아이디가 일치하지 않을 경우
  • 브라우저 새로고침이 실행됩니다.
  • 바깥 if문의 else문 종료
