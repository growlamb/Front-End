변수 (Variable)

변수란 무엇일까?

변수는 합계에 사용할 수 있는 숫자 또는 문자의 일부로 사용할 수 있는 문자열과 같은 값의 container입니다.
그러나 변수에 대한 한 가지 특별한 점은 포함 된 값이 변경 될 수 있다는 점입니다.

변수 선언하기

  • 변수를 사용하려면 먼저 변수 선언을 호출합니다. var, let변수를 호출 할 이름을 입력합니다.
    (예 : let myName; )

변수 초기화하기

  • 먼저 변수 이름을 입력하고 그 뒤에 등호(=), 그 다음 원하는 값을 입력합니다.
    (예 : myName = 'HJ';)

변수 유형

  • Numbers(숫자)
    변수에 숫자를 저장할 수 있습니다. 변수에 수자 값을 지정하면 따옴표가 포함되지 않습니다.
    (예 : let myAge = 20; )
  • Strings(문자열)
    문자열은 텍스트 조각으로 생각하시면 쉽습니다. 변수에 문자열 값을 지정하면 작은 따옴표나 큰 따옴표를 묶어야 합니다. 그렇지 않으면 다른 변수 이름으로 해석합니다.
    (예 : let dolphinGoodbye = 'so long and thank for all the fish'; )
  • Booleans(불 방식의)
    불 방식의은 참과 거짓의 값입니다. 그들은 두개의 값을 가지고 있습니다. true 또는 fals 입니다. 이들은 일반적으로 조건을 테스트하는 데 사용됩니다.
    (예 : let iAmAlive = true; )
  • Arraye(배열)
    배열은 여러값이 대괄호로 묶이고 쉼표로 구분 된 단일 개체입니다.
    (예 : let myNameArray = [ 'ABC', 'DEF', 'GHI'])
  • Objects(물체)
    실체 물체를 모델링하는 코드 구조입니다. 나태내는 물체의 너비, 길이, 및 높에 대한 정보를 포함하는 간단한 물체를 가질 수 있습니다.
    (예 : let cat = { name : 'mall', breed : 'Russian Blue' }; )
