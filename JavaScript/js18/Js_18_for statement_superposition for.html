반복문(For Statement) 중첩 for문

중첩 for문은 for문 안에 for문을 사용하는 것을 말합니다.

중첩 for문 기본형

for(초기값; 조건식; 증감식){
for(초기값; 조건식; 증감식){
실행문;
}}

  • 자바스크립트를 이용해 3행 5열의 표을 만들 경우, 1행씩 행이 만들어질 때마다 5개의 열을 만들어야 할 경우 사용됩니다.


중첩 for문 예시

이번 예시는 for문을 사용해 1행 1열부터 3행 2열까지 출력해 바깥쪽 for문은 행만큼 증가되고, 안쪽 for문은 열만큼 증가됩니다.
for(var a=1; a<=3; a++){
for(var b=1; b<=2; b++){
document.write(a+ "행"+b+"열","<br />");
}}

  • 3행(3번 반복)
  • 2열(2번 반복)
  • 3행이 생성되며 한 행씩 2열이 생성됩니다.
