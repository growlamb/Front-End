반복문(For Statement) for문

for문은 반복문으로 조건식을 만족할 때까지 특정 실행문을 반복해 실행힙니다.
사용용도는 while문과 같지만, while문보다 사용이 편해 사용빈도가 높은편입니다.

for문 기본형

for(초기값; 조건식; 증감식){
실행문;
}


for문 예시

for(var a=1; a<=10; a++){
document.write("반복"+a, "<br />");
}
  • 중괄호 내에 실행문을 10번 반복합니다.
  • 변수가 a가 1씩 증가돼 10번 출력합니다.
  • 변수 a에 초기값 1을 저장합니다.
    조건식을 만족할 경우 중괄호에 실행문을 실행합니다.
    증가 연사자로 변수 a의 값이 1만큼 증가됩니다.
    다시 조건식 만족 여부를 검사한 후, 중괄호에 실행문을 실행할지 아니면 종료할지 결정합니다.
