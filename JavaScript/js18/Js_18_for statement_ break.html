반복문(For Statement) break문

break문은 반복문인 while문 또는 for문에서 break문을 실행하면 조건식과 상관없이 강제로 반복문을 종료합니다.
break문은 반복문을 강제로 끝낼 때 사용합니다.

break문 기본형1

var 변수=초기값;
while(조건식){
break;
실행문;
증감식;
}

break문 기본형2

for(초기값; 조건식; 증감식){
break;
}


break문 예시

이번 예시는 for문을 이용해 1~10까지 반복되도록 만든 예제에 break문을 사용해 변수 a의 값이 6일 경우 강제로 반복문을 종료합니다.

for(var a=1; a<=10; a++){
if(a==6) break; document.write(i, "<br />");
}
  • for문을 사용해 변수 a의 초기값 1부터 조건식 10보다 작거나 같을 때까지 a의 값을 1씩 증가시켜 중괄호에 있는 실행문을 실행합니다.
  • 1부터 시작해 10이 될 때까지 중괄호의 실행문이 실행되면 10번을 반복합니다.
  • 조건문에 의해 변수 a의 값이 6일 경우에 반복문을 강제로 종료합니다.
