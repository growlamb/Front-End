문자 결합 연산자(Character Concatenation Operator)

문자 결합 연산자란 무엇일까?

문자 결합 연산자는 피연산자가(연산 대상 데이터)가 문자형 데이터입니다. 여러 개 문자를 하나의 문자형 데이터로 결할할 때 사용합니다.

문자 결합 연산자 기본형

  • 문자형 데이터 + 문자형 데이터 = 하나의 문자형 데이터
  • 문자형 데이터 + 숫자형 데이터 = 하나의 문자형 데이터

문자 결합 연산자

var t1="학교종이";
var t2="땡땡땡";
var t3=8282;
var t4="어서어서 모이자";
var result;
result=t1+t2+t3+t4;
document.write(result);

  • 변수 t1에 문자형 데이터가 저장됩니다.
  • 변수 t2에 문자형 데이터가 저장됩니다.
  • 변수 t3에 숫자형 데이터가 저장됩니다.
  • 변수 t4에 문자형 데이터가 저장됩니다.
  • 변수 result에 문자+문자+숫자+문자이므로 하나의 문자로 결합된 데이터가 저장됩니다.
